# karachun-alice
Навык алисы для нашего приложения.

Возможные команды:
* Расскажи сказку
* Куда пойти в Новгороде?
* Хочу больше!

1) Расскажи сказку.
Рассказывает случайную сказку фольклёра Ногорода :)

2) Куда пойти в Новгороде?
Советует одно из мест :)

3) Хочу больше!
Пиарим наше приложение :)
